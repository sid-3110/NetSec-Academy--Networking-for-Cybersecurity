## 🎯 What is This?

NetSec Academy is a **free, single-file HTML learning platform** that teaches networking fundamentals specifically for cybersecurity students. No backend, no login, no ads — just open knowledge.

> Built for absolute beginners. No prior knowledge required.
> Covers the networking foundation needed for SOC Analyst, Penetration Tester,
> Cloud Security Engineer, and every other cybersecurity role.

---

## 📚 16 Complete Modules

| # | Module | Topics |
|---|--------|--------|
| 01 | Introduction to Networking | LAN/WAN/MAN, Network devices |
| 02 | How the Internet Works | Client-Server, DNS, complete request flow |
| 03 | IP Addressing | IPv4, IPv6, subnetting, CIDR |
| 04 | Ports & Protocols | Well-known ports, HTTP/SSH/DNS/FTP |
| 05 | OSI Model | All 7 layers with attack mapping |
| 06 | TCP/IP Model | 4-layer stack, encapsulation |
| 07 | TCP vs UDP Deep Dive | Three-way handshake, SYN flood |
| 08 | DNS Deep Dive | Resolution, record types, attacks |
| 09 | Network Communication Flow | Full request lifecycle |
| 10 | Packet Fundamentals | Headers, payload, Wireshark |
| 11 | Network Traffic Analysis | tcpdump, Wireshark, anomaly detection |
| 12 | Network Architecture | Topologies, enterprise design, VLANs |
| 13 | NAT & Routing | NAT types, routing tables, default gateway |
| 14 | Firewalls & Network Security | Packet filter, stateful, NGFW, WAF |
| 15 | Network Monitoring & Logging | IDS/IPS, SIEM, NetFlow |
| 16 | Network Attacks Overview | Port scan, MITM, ARP spoof, DDoS |
